## What to build

Manually validate tap-control feel on a mobile phone, preferably mobile Safari. 

## Acceptance criteria

- [x] Tapping above, below, left, and right of the snake head feels predictable.
- [x] Reverse tap is ignored and does not cause a surprise turn.
- [x] Continuous swipe still works.
- [x] D-pad still works.
- [x] Page still does not scroll during gameplay.
- [x] Tap feedback is disabled so no triangle section highlight appears.
